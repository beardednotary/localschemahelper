Local Schema Helper

Chrome Extension

Privacy Policy

Last updated: October 30, 2025

This Privacy Policy explains how the Local Schema Helper Chrome extension (“the extension”, “we”, “our”) handles information. In short: the extension is designed to run locally in your browser and does not send your data to external servers.

What the extension does

Local Schema Helper analyzes the webpage you currently have open in Chrome to detect existing structured data (such as JSON-LD and microdata) and to help you generate valid schema markup (for example, LocalBusiness or FAQPage).

Data we collect

We do not collect, store, or transmit personal data.

  • No analytics or tracking is built into the extension.
  • No browsing history is collected.
  • No page content is sent to any remote server.

Local storage

The extension may store some values in Chrome’s storage (for example, a business name or address you manually override in the popup) so you don’t have to retype them on every page. This information stays in your browser and is not shared or synced by us.

Permissions explained

  • activeTab – lets the extension read the current page when you click the icon, so it can detect structured data.
  • host permissions – needed so the content script can run on any site you want to analyze.
  • storage – used only to save your own overrides (like business name/address) locally.

These permissions are used solely to fulfill the extension’s single purpose: helping you audit and generate schema markup.

Third parties

The extension does not sell, rent, or transfer data to third parties. We do not use your data for advertising or for any purpose unrelated to structured data generation.

Children’s privacy

This extension is intended for website owners, SEOs, and developers. It does not target or knowingly collect information from children.

Changes to this policy

If we add optional online features in the future (for example, Pro licensing or cloud template sync), this page will be updated and those features will be clearly described.

Contact

If you have questions about this policy or how the extension works, contact: support@localschemahelper.com
